# Car-Parking-Asisstant-SMIW
Car Parking assistant created using Arduino UNO

Used components:
<br/>
-Arduino Uno Rev3
<br/>
-LCD Screen 2x16
<br/>
-buzzer with 5V generator
<br/>
-ultrasonic sensor HC-SR04 2-200cm
<br/>
-9V battery
<br/>
